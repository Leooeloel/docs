## 2.7.0 - 2020-03-17

> - 优化了 canvas 模式下的书写性能
> - 修复了 canvas 模式下，Safari 浏览器无法渲染图片的问题
> - 修复了动态 PPT 用动画音频播放时部分用户听不到声音的问题
> - 修复了动态 PPT 转换音频是空字符串时会转换失败的问题
> - 修复了动态 PPT 播放音频时闪烁问题
> - 修复了刚进入房间时，光标依然是默认光标的问题

## 2.6.4 - 2020-03-02

> - 修复了文字没有指定字号报错问题
> - 修复了侧边栏播放音频问题
> - 修复了媒体切换页面时重新播放问题
> - 修复了在组合内的媒体无法播放问题

## 2.6.3 - 2020-02-23

> - 新增了：``rejectWhenReadonlyErrorLevel``
